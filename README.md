## General info
This is a web application that provides the basic functions of advertising portals. App is built on CQRS pattern, using MediatR and .NET Core technologies.
## Features
- Adding, editing, deleting announcements
- Differenet managment panel for admin
- Admin can manage all announcements
- Special validation rules for some models
